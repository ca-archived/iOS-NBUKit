NBUFilter Class Reference

Inherits from NSObject
Declared in NBUFilter.h

Overview

An abstract filter object.

Abstracts the underlying filter library to have a uniform way of using filters.

%note%
Note: Will be refactored soon.

Class Methods

filterWithName:type:values:defaultValues:identityValues:attributes:provider:configureFilterBlock:

Create a new NBUFilter of a given type, optionally setting a given name and initial values.

+ (id)filterWithName:(NSString *)name type:(NSString *)type values:(NSArray *)values defaultValues:(NSArray *)defaultValues identityValues:(NSArray *)identityValues attributes:(NSDictionary *)attributes provider:(Class<NBUFilterProvider>)provider configureFilterBlock:(NBUConfigureFilterBlock)block

Parameters

name

An optional filter name.

type

The desired filter type.

values

The optional initial values.

defaultValues

The default filter values.

identityValues

The filter values that don’t modify the input.

attributes

A dictionary describing in more detail the input values range, type, etc.

provider

The associated NBUFilterProvider capable of handling the filter.

block

The block to be called to retrieve a configured concreteFilter.

Discussion

%note%
Note: Prefer NBUFilterProvider methods.
