NBURotatingView Protocol Reference

Conforms to NSObject
Declared in NBURotatingViews.h

Overview

A protocol for views that listen to device orientation changes to rotate accordingly.

%note%
Note: Usually used only with view controllers that do not rotate by themselves.

setInterfaceOrientation:animated:

Programatically try rotate the view.

Parameters

The desired interface orientation.

animated

Discussion

If orientation is not among the supportedInterfaceOrientations then
no rotation will take place.
